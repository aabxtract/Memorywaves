# **App Name**: Memory Weavers

## Core Features:

- Connect Wallet: Allow users to connect their Web3 wallet (MetaMask, etc.) to interact with the dApp.
- Submit Memory Fragment: Enable users to submit short memory fragments (max 200 chars) to be stored on-chain.
- Store Fragment On-Chain: Store the memory fragment securely on the blockchain.
- AI-Powered Similarity Clustering: Employ a tool based on an AI model (OpenAI or local) to generate 'similarity clusters' by keyword similarity, and sentiment analysis, plus a bit of randomness. Note: this will be a tool
- Mint Memory Chain NFT: Allow users to mint a selected memory chain as a generative NFT art piece.
- View Memory Map: Display a global Memory Map showing all clusters, where each cluster is a memory chain, bubble size indicates fragment number, and hover reveals sample fragments.
- Chain Page Display: Show all fragments within a selected memory chain on a dedicated page.

## Style Guidelines:

- Primary color: Light pastel purple (#D8B4FE) to evoke a dreamy and ethereal feeling, contrasting against the dark background.
- Background color: Dark navy blue (#1A202C) to create a soft and minimal user interface with good contrast.
- Accent color: Soft lavender (#C4A3E3) to complement the primary color and add visual interest to key elements such as interactive buttons and links.
- Body and headline font: 'Inter' sans-serif font for a modern, neutral look suitable for both headlines and body text.
- Use soft, minimalist icons to represent different features and functions, aligning with the dreamy aesthetic.
- Employ a clean and spacious layout with ample white space to ensure a calming and uncluttered user experience.
- Subtle animations, like smooth transitions and gentle fades, to create a seamless and immersive experience.
